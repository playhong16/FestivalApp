# FestivalApp

## 프로젝트 소개
여행을 갈 때, 저의 주 관심사는 맛집 또는 지역 명소입니다.<br>
하지만 우리나라에서는 지역별로 생각보다 많은 축제가 열리고 있습니다.<br>
이 프로젝트는 제가 여행을 갔을 때, 지역 축제를 확인하고 꼭 가보고 싶은 축제를 저장하기 위해 만든 앱입니다.

## 프로젝트 기간
2024.01.04 ~ 2024.01.

## 구현 기능
### 조회 및 검색
축제 목록을 조회하고, 검색을 통해 원하는 축제 정보를 조회합니다.

### 상세보기
축제에 대한 상세 정보를 보여줍니다.

### 저장
관심 있는 축제를 저장하고, 저장한 목록을 보여줍니다.
